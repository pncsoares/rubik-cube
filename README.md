# Setup

## Clone repository

Create and go to the directory where you want to place the repository

```bash
  cd my-directory
```

Clone the project

```bash
  git clone https://github.com/pncsoares/rubik-cube.git
```

Go to the project directory

```bash
  cd rubik-cube
```
